Show local packages:
	npm list
	npm list -g

Find Local version:
	npm -v <package>

Show newest available version:
	npm view <package-name> version
	npm view [package-name] dist-tags

Un/Install:
	npm install <package name>
	npm install <package name> -g
	npm uninstall <package name>
